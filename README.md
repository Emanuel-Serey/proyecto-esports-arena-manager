# Proyecto eSports Arena Manager

Proyecto desarrollado para la asignatura FullStack 1, basado en una arquitectura de microservicios con Spring Boot.

## Microservicios

- user-service
- game-service
- tournament-service
- team-service
- sanction-service
- registration-service
- match-service
- result-service
- ranking-service
- prize-service

## Tecnologías utilizadas

- Java 25
- Spring Boot 4.0.6
- Spring Web
- Spring Data JPA
- H2 Database
- OpenFeign
- Lombok
- Maven
- IntelliJ IDEA
- Git y GitHub
